# SpiderRock Website Redesign

Two compelling design directions ready for review and deployment.

## 🎨 Version 1: Enhanced Polish (Main)
- **Theme**: Clean, refined, modern fintech
- **Features**: Smooth animations, enhanced micro-interactions, polished spacing
- **Files**: `index.html`, `platform.html`, `brokerage.html`, `data.html`, `contact.html`
- **Best for**: Conservative, professional aesthetic

## 🔥 Version 2: Bold Alternative
- **Theme**: Editorial, modern, strong visual hierarchy  
- **Features**: Bolder typography, red accent bars, dramatic layouts
- **Files**: `index-v2.html` (plus other pages can follow same pattern)
- **Best for**: Making a statement, modern edge

## 🚀 Quick Start

1. **Review locally** (optional):
   - Open `index.html` in a browser for Version 1
   - Open `index-v2.html` for Version 2

2. **Deploy to Netlify**:
   - Follow `DEPLOYMENT.md` for step-by-step instructions
   - Takes ~5 minutes from start to live

3. **Compare on live URLs**:
   - Version 1: Main site
   - Version 2: Preview URL (auto-generated by Netlify)

4. **Iterate**:
   - Make changes to files
   - Push to GitHub
   - Netlify auto-deploys (30 seconds)
