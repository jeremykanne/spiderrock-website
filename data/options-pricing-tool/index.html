How It Works

Two Powerful Calculators

Simple Option Calculator

Calculate a single option price or implied volatility for a specific option using SpiderRock data inputs from a simple web-based GUI or programmatically via API. Ideal for checking prices, validating models, and understanding pricing dynamics.

  • Single option price or volatility calculation
  • Web GUI and API access
  • SpiderRock live volatility surface inputs
  • Supports listed and FLEX options

Option Basket Calculator

Calculate prices for multi-leg option strategies. Input multiple legs and receive consolidated analytics including net premium, Greeks, and implied volatility for the entire basket — supports listed and FLEX options.

  • Multi-leg strategy pricing
  • Net premium and consolidated Greeks
  • Basket implied volatility
  • Listed and FLEX options support
