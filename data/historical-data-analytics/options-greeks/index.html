Data Coverage

What's Included in Every Record

Price & Volume

  • Open, High, Low, Close (OHLC)
  • Volume and open interest
  • Bid/ask and bid/ask sizes
  • Last trade price and size
  • Underlying stock price

Greeks & Risk

  • Delta (direction risk)
  • Gamma (delta sensitivity)
  • Vega (volatility sensitivity)
  • Theta (time decay)
  • Rho (interest rate sensitivity)

Volatility & Meta

  • Implied volatility (bid, ask, mid)
  • Surface volatility from SR model
  • Expiration date and DTE
  • Strike, option type (call/put)
  • Exchange, OCC symbology

Data Delivery

Flexible Access Methods

Historical options and Greeks data is available through multiple delivery channels to fit your workflow — from direct API access to bulk flat file downloads.

  • Liberator API — Programmatic access for targeted queries across dates, underlyings, and expirations
  • Cloud Files — Daily flat files delivered to AWS S3 or Azure Blob Storage for bulk processing
  • Data Portal — Web-based exploration and ad-hoc download for research and validation
