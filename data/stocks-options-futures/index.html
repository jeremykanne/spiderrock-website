Overview

Real-Time Market Data Feeds

SpiderRock provides a wide range of raw and normalized low latency multicast feeds covering US equities, options, and futures markets. Our client-side API turns high multicast packet volumes into individual message events at low latencies — available in C++ and C# versions, free of charge for SpiderRock feed consumers.

Raw Exchange Feeds

Direct from exchange source in real-time. Original exchange protocol for maximum fidelity. Every message delivered as the exchange sends it.

Normalized Feeds

Single binary format across all exchanges. FPGA + kernel bypass technology for sub-millisecond processing. 10GB and 40GB fiber connections via Arista switches.

Hosted Solution

Infrastructure hosted in NY4/5 and CH2 data centers. Use SpiderRock C++/C# APIs or host your own software on SpiderRock hardware for a cost-effective managed solution.
