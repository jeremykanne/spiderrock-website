Trading Platform

FIX Gateway Connectivity

Client-side and street-side FIX gateways with dark quote posting, liquidity routing, and RFQ support.

Technology

FIXBase Binary FIX

SpiderRock's FIX gateways are built on FIXBase binary FIX technology, delivering high-performance, low-latency order routing. FIXBase's binary encoding reduces message size and processing overhead compared to traditional ASCII-based FIX, enabling faster order transmission and confirmation.

All algorithmic parameters supported by SpiderRock's routing engine are accessible via FIX, enabling client OMS/EMS systems to drive SpiderRock's full algo suite without requiring custom integrations.

SpiderRock operates FIX gateways hosted within Equinix NY4/5 and CH2 data centers, with connectivity available via cross-connect for minimum latency.

Gateway Types

Client-Side & Street-Side

Client-Side Gateway

SpiderRock operates a client-facing FIX gateway enabling order entry from client OMS and EMS systems. All SpiderRock algo parameters are supported via FIX message tags, giving clients the full power of SpiderRock's routing engine through their existing front-end systems.

  • ✓Standard FIX 4.2/4.4 protocol
  • ✓Full algo parameter support via custom tags
  • ✓Order acknowledgment and execution reports
  • ✓Hosted in NY4/5 and CH2 data centers

Street-Side Gateway

SpiderRock's street-facing FIX gateway connects to exchanges, executing brokers, and other market participants. It also accepts drop copies from client systems and other OMS connections to facilitate compliance and risk requirements with full position visibility.

  • ✓Direct exchange connectivity
  • ✓Executing broker connections
  • ✓Drop copy support for done-away trades
  • ✓Position consolidation for compliance

Capabilities

Advanced Execution Features

Liquidity Posting

Post liquidity to exchange order books and electronic dark pools. SpiderRock manages order state and repricing to maximize fill rates while minimizing market impact.

Dark Quotes

Access and post dark quotes for block execution. SpiderRock's connectivity to dark pools and crossing networks provides access to institutional liquidity outside the displayed market.

RFQ Support

Request-for-quote workflow for complex multi-leg strategies. SpiderRock supports exchange-based RFQ mechanisms for options spreads and other complex orders requiring negotiation.
