Trading Platform

11 Professional Trading Applications

A complete suite of GUI tools for trading, risk monitoring, historical analysis, and system control — no dedicated hardware required.

Institutional Front-End

Everything You Need, Ready to Run

SpiderRock's front-end is comprised of 11 GUI tools across four categories: trading, risk, historical, and control. These tools provide a complete interface to the SpiderRock Platform — from single order entry and basket trading via algorithmic parameters to portfolio risk management and live data visualization.

No dedicated hardware is required. All GUI tools are currently delivered via Citrix, enabling fast deployment and consistent performance without local infrastructure investment.

A fully functional simulated environment is available for testing new strategies and training new users without risk to live positions.

Application Categories

Four Categories, Complete Coverage

Trading

Order entry, strategy management, and execution monitoring tools. Supports single order entry and basket trading via algorithmic parameters, with full order management and status tracking.

  • •Single order entry
  • •Basket strategy trading
  • •Algo parameter configuration
  • •Order management and tracking

Risk

Real-time Greeks, P&L, and position monitoring dashboards. Dynamic hedging tools enable active portfolio management with live market data integration.

  • •Real-time portfolio Greeks
  • •Live P&L attribution
  • •Dynamic hedging tools
  • •Position monitoring by strategy

Historical

Backtesting and historical analytics tools that leverage SpiderRock's deep database of tick-level options data and analytics, generated from the same models used in live trading.

  • •Strategy backtesting
  • •Historical P&L analysis
  • •Trade performance review
  • •Analytics consistency with live data

Control

System administration, account configuration, and audit tools. Manage risk parameters, account permissions, compliance configurations, and system health from a single control center.

  • •Account and permission management
  • •Risk parameter configuration
  • •Compliance record management
  • •System health monitoring
