Trading Platform

Built-In Compliance

Integrated compliance records, position tracking, and regulatory reporting — CAT and Rule 606 ready.

Integration

Compliance That Adapts to Your Requirements

Compliance is a critical component of institutional trading. SpiderRock's platform offers an elegant solution allowing for efficient integration of evolving compliance requirements via GUI tools or SRSE's MySQL API.

The compliance layer captures comprehensive records across positions, locates, restricted lists, and done-away trades. All compliance data is stored in the SR Control live database, accessible in real time via the SRSE API for integration with your compliance systems.

The platform is SOC-II compliant, ensuring your data and operations meet institutional security and availability standards.

Compliance Records

Complete Audit Trail

Positions

Real-time position tracking across all strategies and accounts. Positions are updated on every fill and available immediately via SRSE, giving compliance teams an accurate view of exposure at all times.

Locates

Short locate tracking and management integrated into the execution workflow. Locate balances are checked at order entry and decremented on execution, preventing short sales without proper locate approval.

Restricted Lists

Automated screening against configurable restricted security lists. Orders for restricted securities are blocked at the pre-trade stage, with full logging of blocked order attempts for compliance review.

Done Away Trades

Recording and reporting of trades executed away from SpiderRock. Done-away trade records are integrated into the consolidated position and compliance views for complete portfolio transparency.

Regulatory Reporting

CAT & Rule 606 Ready

SpiderRock's compliance layer includes support for major regulatory reporting requirements. CAT (Consolidated Audit Trail) reporting captures order lifecycle events in the format required by FINRA and the SEC. Rule 606 order routing disclosure reports are generated automatically.

Account configuration supports multiple entity structures, enabling broker-dealer and investment advisor clients to maintain proper separation of accounts while using a shared execution infrastructure.

CAT Reporting

Consolidated Audit Trail reporting support for all order types and lifecycle events.

Rule 606 Disclosure

Automated quarterly order routing disclosure report generation.

MySQL API Access

All compliance data accessible via SRSE MySQL API for integration with external compliance systems.
