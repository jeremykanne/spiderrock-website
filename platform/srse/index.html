SpiderRock Structured Engine

Six live databases with bulk query and insert capabilities — the data backbone of the SpiderRock Platform.

Architecture

The Engine Behind the Platform

SRSE (SpiderRock Structured Engine) is SpiderRock's custom high-performance live database engine. It is comprised of six MySQL live databases and enables bulk querying and insert operations to interact with the platform's core data in real time.

Clients integrate programmatically with SpiderRock via SRSE for trading, risk, data, and compliance. The MySQL interface means integration is straightforward — any environment with MySQL client libraries can connect. Client libraries are also available in C++ and C#.

Bulk select and insert operations enable high-throughput programmatic workflows, from strategy engines that query thousands of options simultaneously to risk systems that update positions across the entire portfolio on every trade.

Live Databases

Six Specialized Data Stores

L

SR Live

Real-time market data: quotes, prints, and order book data across all listed US equity, options, and futures markets. Updated on every exchange message.

T

SR Theo

Theoretical pricing: implied volatilities, Greeks, and volatility surface prices updated on every option tick and at regular intervals. The authoritative source for SpiderRock analytics.

A

SR Analytics

Advanced analytics: alpha probabilities, trade performance metrics, and historical context drawn from SpiderRock's multi-year data archive.

R

SR Risk

Portfolio risk: real-time Greeks aggregation, P&L calculations, and margin computations. Updated continuously as positions change and markets move.

Tr

SR Trade

Trade records: execution history, fill details, and trade analytics. Every fill is captured with full context including analytics at time of execution.

C

SR Control

System state: account configuration, positions, locates, restricted lists, and operational controls. The primary interface for compliance and system management.

API Capabilities

Bulk Select & Insert

SRSE's bulk operation capabilities enable strategy engines, risk systems, and analytics platforms to query large datasets efficiently. A single bulk select can retrieve quotes, analytics, and risk data across thousands of options simultaneously.

Bulk insert allows client systems to push configurations, compliance records, and done-away trades into the platform programmatically, enabling tight integration between SpiderRock and client-side systems.

MySQL-Compatible Interface

Connect with any MySQL client. No proprietary drivers or custom protocols required.

C++ & C# Client Libraries

High-performance native libraries for latency-sensitive applications.

Real-Time Data Access

Query live databases for current-state data with sub-second freshness across all six stores.
